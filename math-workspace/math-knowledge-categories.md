# 数学知识多维度分类体系

> 分类时间：2026-04-28 | 模型：deepseek-v4-flash

---

## 维度一：学科知识板块（横向领域）

### 1. 数与代数
- **数的认识与运算**：小数/分数/百分数/有理数/实数/复数
- **代数式与等式**：整式/分式/方程（一元一次→一元二次→二元一次方程组）
- **函数**：一次函数→幂/指数/对数函数→三角函数→数列
- **数量关系与模型**：行程/工程/价格/增长率等应用题

### 2. 图形与几何
- **基本图形认识与测量**：多边形面积→圆→圆柱圆锥→三角形/四边形性质→立体几何
- **位置与变换**：数对坐标→轴对称/旋转/平移→向量运算
- **解析几何**：直线/圆方程→椭圆/双曲线/抛物线方程

### 3. 统计与概率
- **数据收集与整理**：统计图/平均数→方差/标准差→回归分析
- **概率**：可能性→列举法→条件概率→二项/超几何/正态分布

### 4. 综合与实践
- **数学思想方法**：转化与化归/数形结合/分类讨论/函数与方程/整体思想
- **逻辑推理与证明**：几何证明→命题逻辑→综合法/分析法/反证法/数学归纳法

## 维度二：思维层级（纵向深度）

| 层级 | 核心思维 | 代表知识点 |
|------|---------|-----------|
| L1 基础运算层 | 模仿记忆/程序操作 | 运算法则、公式记忆、特殊角三角函数值 |
| L2 概念理解层 | 本质理解/多表示关联 | 函数概念、图形性质、概率含义 |
| L3 逻辑推理层 | 演绎推理/证明论证 | 几何证明、全等判定、充分必要条件 |
| L4 问题建模层 | 抽象建模/综合应用 | 列方程解应用题、导数优化、统计建模 |
| L5 创新思维层 | 反思优化/一题多解/探究 | 开放性问题、多解比较、错题反思 |

---

## 维度三：核心素养主线（螺旋递进）

```
数感 → 式感 → 方程思想 → 函数思想
几何直观 → 空间观念 → 向量思维 → 解析几何思维
数据分析观念 → 随机思维 → 统计推断能力
推理能力 → 模型思想 → 数学建模
```

---

## 维度四：解题策略与考试热点

| 类型 | 典型内容 |
|------|---------|
| 高频计算模型 | 方程求解/函数最值/数列求和/向量运算 |
| 几何证明模型 | 全等相似/平行垂直/圆综合/空间垂直平行 |
| 函数与导数综合 | 零点/单调性/极值最值/恒成立/切线 |
| 解析几何综合 | 定点定值/弦长中点/最值范围/向量法 |
| 概率统计综合 | 分布期望/统计推断/独立性检验/回归分析 |
| 不等式专题 | 基本不等式/绝对值不等式/柯西不等式/放缩技巧 |

## 教与学建议

**教师**：
- 备课：先按维度一定领域，再按维度二设计梯度教学
- 复习：用维度三串联同素养知识构建网络
- 考前：重点参考维度四总结解题模型

**学生**：
- 学新知：尝试按维度一归入已有框架
- 做练习：识别题目对应维度二的哪个层级
- 整理错题：区分是知识短板还是思维漏洞
