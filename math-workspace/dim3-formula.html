维度三：核心素养主线（螺旋递进）

四大主线：数感→式感→方程→函数 / 几何直观→空间观念→向量思维→解析几何 / 数据分析→随机思维→统计推断 / 推理能力→模型思想→数学建模

主线1 数感→式感→方程→函数

数感阶段

  • 小数乘除法(五年级): $a \times b = c$，因数小数位数之和；常见公式：无
  • 分数乘除法(六年级): $\frac{a}{b} \times \frac{c}{d} = \frac{ac}{bd}$；$\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}$
  • 正负数(初一): $a + (-b) = a - b$；$(-a) \times (-b) = ab$
  • 有理数运算(初一): 加法交换律 $a + b = b + a$；结合律 $(a + b) + c = a + (b + c)$
  • 百分数(六年级): $p\% = \frac{p}{100}$；$a \times p\% = \frac{ap}{100}$
  • 比和比例(六年级): $a:b = \frac{a}{b}$；$\frac{a}{b} = \frac{c}{d} \Rightarrow ad = bc$

式感阶段

  • 幂运算(初一): $a^m \cdot a^n = a^{m+n}$；$(a^m)^n = a^{mn}$
  • 整式加减(初一): 合并同类项 $ax + bx = (a+b)x$；去括号 $-(a-b) = -a + b$
  • 乘法公式(初二): $(a+b)^2 = a^2 + 2ab + b^2$；$(a-b)(a+b) = a^2 - b^2$
  • 因式分解(初二): $a^2 - b^2 = (a-b)(a+b)$；$a^2 + 2ab + b^2 = (a+b)^2$

方程阶段

  • 一元一次方程(初一): $ax + b = 0 \Rightarrow x = -\frac{b}{a}$
  • 二元一次方程组(初一): 加减消元法 / 代入消元法；常见公式：无
  • 一元二次方程(初三): $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$；$\Delta = b^2 - 4ac$

函数阶段

  • 一次函数(初二): $y = kx + b$；$k = \frac{y_2 - y_1}{x_2 - x_1}$
  • 二次函数(初三): $y = ax^2 + bx + c$；顶点 $(-\frac{b}{2a}, \frac{4ac - b^2}{4a})$
  • 反比例函数(初二): $y = \frac{k}{x}$；$k = xy$
  • 指数函数(高一): $y = a^x \ (a > 0, a \neq 1)$；单调性：$a > 1$ 递增，$0 < a < 1$ 递减
  • 对数函数(高一): $y = \log_a x \ (a > 0, a \neq 1)$；$\log_a(MN) = \log_a M + \log_a N$；换底公式 $\log_a b = \frac{\log_c b}{\log_c a}$
  • 三角函数(高一): $\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$；$\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$；$\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}$；$\sin 2\alpha = 2\sin\alpha\cos\alpha$；$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha$
  • 数列(高中): 通项公式 $a_n$；前 $n$ 项和 $S_n$
  • 等差数列(高中): $a_n = a_1 + (n-1)d$；$S_n = \frac{n(a_1 + a_n)}{2}$
  • 等比数列(高中): $a_n = a_1 \cdot q^{n-1}$；$S_n = \frac{a_1(1 - q^n)}{1 - q}$
  • 导数(高中): $(x^n)' = nx^{n-1}$；$(e^x)' = e^x$；$(\ln x)' = \frac{1}{x}$；$(uv)' = u'v + uv'$

主线2 几何直观→空间观念→向量思维→解析几何

几何直观

  • 数轴(初一): $|a-b|$ 表示距离
  • 相反数(初一): $a + (-a) = 0$
  • 绝对值(初一): $|a| = \begin{cases} a & (a \ge 0) \\ -a & (a < 0) \end{cases}$
  • 全等三角形(初二): SSS / SAS / ASA / AAS / HL
  • 轴对称(初二): 关于 $x$ 轴或 $y$ 轴对称，坐标变号
  • 等腰三角形(初二): 等边对等角，三线合一

空间观念

  • 平行四边形(初二): $S = ah$；对边平行相等
  • 圆(初三): $C = 2\pi r$；$S = \pi r^2$；弧长 $l = \frac{n}{360} \times 2\pi r$
  • 勾股定理(初二): $a^2 + b^2 = c^2$；常用勾股数 $(3,4,5)$
  • 锐角三角函数(初三): $\sin A = \frac{a}{c}$；$\cos A = \frac{b}{c}$；$\tan A = \frac{a}{b}$；$\sin^2 A + \cos^2 A = 1$

向量思维

  • 空间向量(高中): 向量坐标 $(a_1, a_2, a_3)$；点积 $a \cdot b = a_1 b_1 + a_2 b_2 + a_3 b_3$；模长 $|a| = \sqrt{a_1^2 + a_2^2 + a_3^2}$

解析几何

  • 圆锥曲线(高中): 椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$；双曲线 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$；抛物线 $y^2 = 2px$；离心率 $e = \frac{c}{a}$

主线3 数据分析→随机思维→统计推断

数据分析

  • 百分数(六年级): $p\% = \frac{p}{100}$
  • 比和比例(六年级): $a:b = c:d \Rightarrow ad = bc$
  • 数据的收集与整理(初一/初二): 平均数 $\bar{x} = \frac{1}{n}\sum_{i=1}^n x_i$；方差 $s^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \bar{x})^2$；标准差 $s = \sqrt{s^2}$；中位数、众数反映数据中心趋势

随机思维

  • 概率初步(初三): $P(A) = \frac{m}{n}$；$P(\Omega) = 1$；$P(\varnothing) = 0$
  • 排列组合(高中): $A(n,m) = \frac{n!}{(n-m)!}$；$C(n,m) = \frac{n!}{m!(n-m)!}$；$C(n,m) = C(n,n-m)$

统计推断

  • 二项分布(高中): $P(X = k) = C(n,k) p^k (1-p)^{n-k}$；$E(X) = np$；$D(X) = np(1-p)$
  • 正态分布(高中): 正态曲线 $\varphi(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$；$3\sigma$ 原则 $P(\mu - 3\sigma < X < \mu + 3\sigma) \approx 99.73\%$

主线4 推理能力→模型思想→数学建模

推理能力

  • 乘法公式(初二): $(a+b)^2 = a^2 + 2ab + b^2$；$(a-b)^2 = a^2 - 2ab + b^2$；$(a-b)(a+b) = a^2 - b^2$
  • 因式分解(初二): $a^2 - b^2 = (a-b)(a+b)$；$a^2 + 2ab + b^2 = (a+b)^2$；十字相乘法 $x^2 + (a+b)x + ab = (x+a)(x+b)$
  • 基本不等式(高中): $\frac{a+b}{2} \ge \sqrt{ab}$；$a^2 + b^2 \ge 2ab$；$\frac{a+b+c}{3} \ge \sqrt[3]{abc}$
  • 绝对值不等式(高中): $|x| < a \Rightarrow -a < x < a$；$|x| > a \Rightarrow x > a$ 或 $x < -a$；$|x \pm y| \le |x| + |y|$
  • 柯西不等式(高中): $(a_1^2 + a_2^2)(b_1^2 + b_2^2) \ge (a_1 b_1 + a_2 b_2)^2$；等号成立 $\frac{a_1}{b_1} = \frac{a_2}{b_2}$
  • 数学归纳法(高中): 第一步验证 $n=1$ 时成立；第二步假设 $n=k$ 成立，推证 $n=k+1$ 成立
  • 证明方法: 综合法（由因导果）、分析法（执果索因）、反证法（否定结论导出矛盾）

模型思想

  • 一元二次方程(初三): $ax^2 + bx + c = 0 \Rightarrow x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$；韦达定理 $x_1 + x_2 = -\frac{b}{a}$，$x_1 \cdot x_2 = \frac{c}{a}$
  • 二元一次不等式(高中): $ax + by + c > 0$ 表示平面区域；线性规划求最优解

数学建模

  • 数学建模(高中): 建立函数模型 $y = kx + b$；增长模型 $y = a \cdot e^{bx}$
  • 导数模型: $(f(x)g(x))' = f'(x)g(x) + f(x)g'(x)$；优化问题用导数求极值
  • 课题学习(各年级): 研究方法论：观察 → 假设 → 验证
