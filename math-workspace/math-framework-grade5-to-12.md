# 小学数学（五年级～六年级）

## 五年级

### 知识模块表格
| 模块 | 主要内容 |
|------|----------|
| 小数乘法与除法 | 小数乘整数、小数乘小数、积的近似数、小数除以整数、一个数除以小数、商的近似数、循环小数 |
| 位置 | 用数对确定位置 |
| 简易方程 | 用字母表示数、方程的意义、解方程、列方程解决问题 |
| 多边形的面积 | 平行四边形的面积、三角形的面积、梯形的面积、组合图形的面积 |
| 可能性 | 事件发生的可能性大小，游戏规则公平性 |

### 核心知识点
- 小数乘除法的计算法则及与整数乘除法的联系
- 数对（列，行）表示位置
- 等式的性质与解方程的基本方法
- 平行四边形、三角形、梯形面积公式的推导
- 用割补法求组合图形面积

### 重要公式
- 小数乘法：先按整数乘法算出积，再点小数点
- 小数除法：除数扩大多少倍，被除数也扩大相同倍数
- 平行四边形面积：$S = a h$
- 三角形面积：$S = \frac{1}{2} a h$
- 梯形面积：$S = \frac{1}{2}(a + b) h$

- 反比例关系：$xy = k$（$k$为常数）

| 数据的收集与整理 | 统计调查、条形图、扇形图、折线图、直方图 |

### 核心知识点
- 有理数的分类与运算规则（符号法则）

  - $\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$
  - $\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}$
  - $\sin 2\alpha = 2\sin\alpha\cos\alpha$
  - $\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 2\cos^2\alpha - 1 = 1 - 2\sin^2\alpha$
- 向量坐标运算：$\vec{a}=(x_1,y_1)$，$\vec{b}=(x_2,y_2)$，$\vec{a}\cdot\vec{b}=x_1x_2+y_1y_2$

- 复数：$(a+bi)(c+di)=(ac-bd)+(ad+bc)i$
- 圆柱体积：$V=\pi r^2 h$，圆锥体积：$V=\frac{1}{3}\pi r^2 h$，球体体积：$V=\frac{4}{3}\pi r^3$

### 重要公式
- 定积分微积分基本定理：$\int_a^b f(x) \, dx = F(b) - F(a)$，其中 $F'(x)=f(x)$
- 正弦定理：$\frac{a}{\sin A} = 2R$
- 余弦定理：$a^2 = b^2 + c^2 - 2bc\cos A$
- 三角形面积：$S = \frac{1}{2}ab\sin C = \frac{abc}{4R} = \frac{1}{2}(a+b+c)r$
- 柯西不等式（二维）：$(a^2+b^2)(c^2+d^2) \geq (ac+bd)^2$
- 绝对值三角不等式：$||a|-|b|| \leq |a \pm b| \leq |a| + |b|$
- 弦长公式：$|AB| = \sqrt{1+k^2} \cdot \frac{\sqrt{\Delta}}{|a|}$（联立直线与圆锥曲线，$\Delta$为判别式）
- 点到直线距离公式（二维）：$d = \frac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}$
- 正态分布：$X \sim N(\mu, \sigma^2)$，$P(\mu-\sigma < X < \mu+\sigma) \approx 0.6827$，$P(\mu-2\sigma < X < \mu+2\sigma) \approx 0.9545$，$P(\mu-3\sigma < X < \mu+3\sigma) \approx 0.9973$
